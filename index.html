📐 Mathematical Equations

Forward Propagation:

z₁ = X · W₁ + b₁
a₁ = σ(z₁) // activation function
z₂ = a₁ · W₂ + b₂
ŷ = sigmoid(z₂) // for classification

Loss Function:

L = -1/m Σ [y·log(ŷ) + (1-y)·log(1-ŷ)] // Binary Cross-Entropy

Backward Propagation (Gradients):

∂L/∂z₂ = ŷ - y
∂L/∂W₂ = a₁ᵀ · ∂L/∂z₂
∂L/∂b₂ = Σ(∂L/∂z₂)
∂L/∂a₁ = ∂L/∂z₂ · W₂ᵀ
∂L/∂z₁ = ∂L/∂a₁ ⊙ σ'(z₁) // element-wise product
∂L/∂W₁ = Xᵀ · ∂L/∂z₁
∂L/∂b₁ = Σ(∂L/∂z₁)

Weight Update (Gradient Descent):

W₁ ← W₁ - α · ∂L/∂W₁
W₂ ← W₂ - α · ∂L/∂W₂
b₁ ← b₁ - α · ∂L/∂b₁
b₂ ← b₂ - α · ∂L/∂b₂
where α = learning rate
